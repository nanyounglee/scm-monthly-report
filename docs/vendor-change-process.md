# 협력사 변경 프로세스 표준 문서

> 작성: 외주생산파트 | 최초 작성: 2026-06 | 버전: v1.0

---

## 1. 목적

외주생산파트 협력사 변경 시 **일관된 기준과 절차**로 검토·실행·아카이브가 이루어지도록 표준 프로세스를 정의한다.  
담당자 개인 경험에 의존하지 않고, 누구나 동일한 플로우로 변경을 진행할 수 있도록 한다.

---

## 2. 적용 범위

- 기존 협력사 → 신규 협력사 이관 (전면 교체)
- 기존 협력사 유지 + 신규 협력사 추가 (복수 운영)
- 협력사 내 공정 변경 (동일 업체, 공정 재배치)
- 신규 협력사 발굴 및 등록

---

## 3. 변경 사유 분류

| 코드 | 사유 | 예시 |
|---|---|---|
| Q | 품질 기준 미달 | 반복 품질 이슈, 등급 미달 |
| C | 단가 경쟁력 저하 | 단가 상승, 대체 업체 단가 우위 |
| D | 납기 대응 부족 | 리드타임 과다, 긴급 대응 불가 |
| O | 운영 효율 문제 | 소통 불량, 수동 프로세스, 시스템 미연동 |
| S | 공급망 전략 조정 | 티어 재편, 복수 협력사 구조 구축 |
| R | 리스크 관리 | 단일 소싱 위험, 창고 이전, 폐업 우려 |

---

## 4. 6단계 변경 검토 프로세스

```
[1단계] 변경 검토 요청
      ↓
[2단계] 현황 파악 (QCD 현황 분석)
      ↓
[3단계] 후보 협력사 평가
      ↓
[4단계] 샘플 테스트
      ↓
[5단계] 최종 판정 및 보고
      ↓
[6단계] 이관 실행 및 모니터링
```

---

### 1단계 — 변경 검토 요청

**트리거 조건** (하나 이상 해당 시 검토 개시):
- 동일 협력사에서 동일 유형 이슈가 **월 3건 이상** 반복
- 단가 비교 시 경쟁 업체 대비 **15% 이상** 차이 발생
- 납기 지연이 **월 2회 이상** 발생
- 파트 내 전략적 판단으로 변경 검토 필요

**담당자 액션**:
- 변경 검토 Airtable 뷰에 항목 등록
- 변경 사유 코드 입력 (Q/C/D/O/S/R)
- 검토 담당자 지정

---

### 2단계 — 현황 파악 (QCD 현황 분석)

**분석 항목**:

| 항목 | 기준 | 데이터 출처 |
|---|---|---|
| 품질이슈 발생률 | 최근 3개월 이슈건/발주건 | KPI 집계 시트 |
| 실패비용 | 최근 3개월 반품·재작업 비용 | KPI_매입액 |
| 평균 리드타임 | 발주일→입고일 평균 | 에어테이블 TASK |
| 긴급 발주 비율 | 긴급건/전체발주건 | KPI 집계 시트 |
| 단가 비교 | 현재 단가 vs 시장 단가 | 코스트베이스 |
| 커뮤니케이션 응답성 | 주관적 평가 (1~5점) | 담당자 평가 |

**산출물**: QCD 현황 분석표 (협력사 변경 검토 대시보드에 입력)

---

### 3단계 — 후보 협력사 평가

**QCD 가중치 평가표**:

| 평가 영역 | 가중치 | 평가 항목 |
|---|---|---|
| 품질 (Q) | 30% | 샘플 품질, 품질관리 체계, 이슈 대응력 |
| 원가 (C) | 25% | 단가 경쟁력, MOQ 조건, 추가 비용 구조 |
| 납기 (D) | 25% | 표준 리드타임, 긴급 대응 가능 여부, 케파 |
| 운영 (O) | 20% | 소통 방식, 시스템 연동, 담당자 전문성 |

**평가 점수 기준** (각 항목 1~10점):
- 최종 가중 평균 = Q×0.3 + C×0.25 + D×0.25 + O×0.2

**확인 필수 항목**:
- 사업자등록증 및 기본 계약 가능 여부
- 보유 설비 및 생산 케파
- 주요 거래처 레퍼런스
- 하도급법 계약 가능 여부
- 소량 주문 대응 가능 여부 (100개 미만)
- 연질 UV·특수 인쇄 등 공정별 특이 조건

---

### 4단계 — 샘플 테스트

**샘플 요청 기준**:
- 실제 발주 예정 제품과 동일 사양으로 샘플 제작 요청
- 샘플 수량: 최소 3~5개
- 평가 기준: 신시어리 품질 등급표 기준 적용

**샘플 평가 체크리스트**:

| 항목 | 확인 내용 | 판정 |
|---|---|---|
| 인쇄 품질 | 색상 재현율, 선명도, 번짐 여부 | ○/△/× |
| 소재/마감 | 표면 마감, 코팅 상태, 재질 일치 | ○/△/× |
| 치수 정확도 | 실측 사이즈 vs 도면 사이즈 | ○/△/× |
| 수량 | 요청 수량 대비 납품 수량 | ○/△/× |
| 납기 | 샘플 요청일 → 수령일 | 일 수 기록 |
| 패키징 | 보호 포장 상태 | ○/△/× |

---

### 5단계 — 최종 판정 및 보고

**판정 기준**:

| 가중 평균 점수 | 판정 | 조치 |
|---|---|---|
| 6.6점 이상 | ✅ 변경 권고 | 보고서 작성 후 실행 |
| 4.1 ~ 6.5점 | 🔶 추가 검토 | 재샘플 또는 조건 협의 후 재판정 |
| 4.0점 이하 | ❌ 변경 보류 | 현 협력사 유지 또는 다른 후보 탐색 |

**보고서 포함 내용**:
- 변경 사유 및 배경
- 현황 분석 요약 (QCD 비교표)
- 후보 협력사 평가 결과
- 샘플 테스트 결과
- 리스크 및 대응 방안
- 이관 실행 일정

**보고서 양식**: `협력사_변경_보고서_[품목명]_[날짜].md`  
**아카이브 위치**: 협력사 변경 검토 대시보드 → 보고서 탭

---

### 6단계 — 이관 실행 및 모니터링

**이관 실행 체크리스트**:

| 항목 | 담당 | 기한 |
|---|---|---|
| 에어테이블 협력사 DB 변경 (아이템별 공정 협력사) | 해당 담당자 | 이관일 전 |
| 코스트베이스 단가 업데이트 | 해당 담당자 | 이관일 전 |
| 제품정의서 업데이트 | 해당 담당자 | 이관일 전 |
| 가격공지채널 공지 (단가 변경 시) | 파트장 확인 후 | 이관일 당일 |
| 기존 협력사 거래 종료 안내 (필요 시) | 해당 담당자 | 이관 후 1주 이내 |
| 신규 협력사 발주서 1회 검토 | 파트장 | 첫 발주 시 |

**초기 모니터링 기간**: 이관 후 **4주** (첫 발주 3건 또는 1개월, 빠른 것 기준)

**모니터링 항목**:
- 납기 준수율
- 품질 이슈 발생 여부
- 단가 실제 적용 확인
- 긴급 대응 가능 여부 실검증

**이관 완료 선언 조건**: 모니터링 기간 중 이슈 없을 경우 "이관 완료" 처리  
→ 협력사 변경 검토 대시보드 상태 "모니터링" → "완료" 변경

## 6. 자주 묻는 질문

**Q. 샘플 비용은 누가 부담하나요?**  
A. 원칙적으로 발주처(신시어리) 부담. 단, 협력사가 무상 제공하는 경우 수용 가능. 첫 거래 시 무상 샘플 요청 권장.

**Q. 변경 보류 판정 후 재검토는 가능한가요?**  
A. 가능. 협력사 측 조건 개선(단가 인하, 샘플 재제작 등) 시 3개월 이후 재검토 가능.

**Q. 기존 협력사와 신규 협력사를 동시 운영해도 되나요?**  
A. 가능. 초기 이관 시 긴급 대응 목적으로 기존 협력사를 백업으로 유지하는 것을 권장.

**Q. 인수인계 표준문서는 어디에서 확인하나요?**  
A. 에어테이블 협력사 DB > 인수인계 표준문서 뷰 참조. 양식은 김영준 담당자 작성 기준 적용.
